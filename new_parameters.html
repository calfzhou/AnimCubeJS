AnimCube Customization - New Parameters

This page is the list of new parameters added to AnimCube as part of the conversion to JavaScript, there are many more in Josef Jelinek's AnimCube Documentation.



supercube set to 1 for supercube
gabbacolors set to 1 for Gabbasoft colors as shown on the second cube below
superfacelets layout is same as facelets parameter, values are:
0=normal, 1=clockwise, 2=half-twist, 3=counter-clockwise


The gabbacolors parameter can also be used for the regular cube as shown on the second cube below, this option also makes the borders thinner.




snap
If set to 1 then a partially twisted face will not remain in a position between two corners. The difference can be seen on the following cubes, if you use the mouse to twist a face so it is half-way between two corners then the first cube will remain as is, the second cube will snap to a corner. If the face is twisted less than half way to the next corner then it will snap back to the starting position, if it is twisted more than half way then it will snap to the next corner.




repeat
If set to 0 then pressing Play or Single-Step will not repeat a sequence after it has already been played, this is to make it clear that a sequence is complete. The difference can be seen by single-stepping (second button from right) through the sequence on the two cubes below, the first allows repeat and the the second does not.




clickprogress
If set to 0 then clicking on the progressbar will not advance through the move sequence, this is to avoid inadvertently messing up the playback of a sequence while single-stepping through it. The difference can be seen on the following cubes, if you click on the progress bar on the first cube the sequence will advance, the second one will not.




buttonheight - range 9-25 pixels

textsize - range 5-40 pixels

slidercolor - hex codes

sliderbgcolor - hex codes
