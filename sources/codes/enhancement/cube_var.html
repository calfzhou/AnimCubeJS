Accessing the cube configuration

This web page is an example of how to access the cube configuration from the AnimCubeJS program. View the source code of this page to see that the cube variable is declared prior to loading the AnimCube3mod.js script. The only modification to AnimCube3mod.js was to comment out the declaration of cube which will cause it to use the one in this web page instead.

The cube configuration (facelets) are shown below the cube. Use the mouse to turn some layers, then press the Show Facelets button to see that AnimCubeJS has updated the configuration. Moves made by playing the sequence (using the button bar) will also update the configuration but rotation of the whole cube (via mouse or touch) does not update the cube configuration.
